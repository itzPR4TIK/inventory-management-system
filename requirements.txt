# 📦 Inventory Management System

A professional inventory management application built using **Python**, **SQLite**, and **Streamlit**.

This project provides both:
- ✅ Command Line Interface (CLI)
- ✅ Web Dashboard Interface

Designed to simulate real-world inventory operations used in industrial environments.

---

## 🚀 Features

- Add new products
- View inventory records
- Update product details (name, quantity, price)
- Delete products safely with confirmation
- Interactive web dashboard
- Inventory statistics and overview

---

## 🛠️ Tech Stack

- Python
- SQLite Database
- Streamlit (UI Dashboard)
- Git & GitHub
